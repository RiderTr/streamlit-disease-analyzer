# CDI Analytics Dashboard (Streamlit)

Цей проект — інтерактивний веб-дашборд для аналізу великого набору медичних даних (U.S. Chronic Disease Indicators). Додаток дозволяє фільтрувати дані, знаходити статистику, переглядати інтерактивні графіки та географічні карти, прогнозувати тренди, а також завантажувати готові звіти у форматах CSV та PDF

Інтерфейс побудований на базі сучасного фреймворку **Streamlit** з використанням **Plotly** для візуалізації

# Структура проекту
```
├── U.S._Chronic_Disease_Indicators.csv   <-- Датасет (потрібно завантажити і покласти сюди)
│
├── main.py          <-- Точка входа, настройка FastAPI и эндпоинты
├── data_loader.py   <-- Модуль завантаження, очищення даних та словники розшифровок
├── analytics.py     <-- Математична логіка, агрегація та пошук
├── report.py        <-- Генерація графіків (Matplotlib) та файлів для скачування
└── README.md        <-- Опис проекту
```

# Інструкція із запуску

## Крок 1: Підготовка оточення
Переконайтеся, що у вас встановлено Python (версія 3.8+). Встановіть необхідні бібліотеки за допомогою pip:
```
pip install streamlit pandas plotly fpdf2 numpy
```

## Крок 2: Підготовка даних
Помістіть у папку з кодом вихідний датасет. Файл має називатися строго U.S._Chronic_Disease_Indicators.csv.
https://catalog.data.gov/dataset/u-s-chronic-disease-indicators - сайт з датасетом

## Крок 3: Запуск сервера
Відкрийте термінал у папці з проектом та виконайте команду:
```
streamlit run app.py
```

## Крок 4: Використання (Інтерфейс користувача)

Після успішного старту скрипта у вашому браузері автоматично відкриється вкладка з адресою http://localhost:8501
У лівому боковому меню вам будуть доступні наступні розділи:

- Головна (Досьє хвороби) - швидке зведення по конкретній хворобі, генерація прогнозу на 5 років та завантаження PDF-звіту.
- Топ захворювань - таблиця та графік найпоширеніших проблем зі здоров'ям із можливістю налаштування їх кількості.
- Пошук хвороби - пошук конкретної метрики та її відображення на інтерактивній тепловій карті США.
- Аналіз по штатах - детальний стовпчиковий графік по всіх штатах для обраної хвороби з можливістю експорту сирих даних у CSV.
- Прогресія (Тренди) - лінійний графік зміни захворюваності по роках.
- Вік та Стать - демографічний аналіз з використанням таблиць та кільцевих діаграм Plotly.
- Порівняння трендів - виведення двох хвороб на один графік для порівняння їх динаміки.
